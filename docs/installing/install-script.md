### محتویات فایل دانلود شده

پس از دانلود و خارج کردن فایل از حالت فشرده محتوای فایل به صورت زیر میباشد:

- build.zip
- app.zip
- 18.zip
- 20.zip
- general.pdf (آموزش موارد عمومی مانند ایجاد ایمیل و پایگاه داده)
